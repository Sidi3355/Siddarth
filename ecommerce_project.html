AI-Powered Exoskeleton for Knee Rehabilitation

A wearable exoskeleton using sensors and machine learning to assist knee recovery and improve mobility after injury.

Project Overview

As a football and rugby player, I witnessed how limited rehabilitation access affects semi-pro and amateur athletes. This EPQ explores an AI-enhanced exoskeleton that combines biomechanical engineering with machine learning to provide intelligent, accessible sports injury rehabilitation. The project demonstrates how data-driven technology can bridge the gap between professional-grade equipment and the needs of everyday athletes recovering from lower-limb injuries.

The Challenge

Only 43.4% of ACL injury patients return to their previous sport level. Semi-pro and amateur athletes face mounting NHS waiting times, limited access to specialized equipment, and generic braces that provide no feedback on recovery progress. Traditional solutions offer passive support without adapting to individual recovery stages or providing data-driven insights. This project asked whether an AI-enhanced exoskeleton could democratize effective rehabilitation through intelligent hardware and machine learning.

The Solution

I developed a dual-prototype system merging intelligent hardware with machine learning. The physical exoskeleton weighs 400g and features curved polycarbonate braces, adjustable straps fitting the 5th-95th percentile, motor-driven gait assistance, and elastic plates for knee extension. The AI system includes a deep neural network achieving 100% accuracy in classifying movement patterns, an LSTM autoencoder assessing rehabilitation progress (0.043 testing loss), and an Arduino sensor platform streaming real-time data to an external dashboard.

Key Features

CNC-routed polycarbonate braces with heat-formed ergonomic curves
3D-printed nylon hinges and modular elastic plate system
MMA8451 accelerometer and ultrasonic distance sensor integration
Edge Impulse classification model for activity recognition
LSTM autoencoder for injury status prediction
Live data streaming via Arduino Cloud
Drop-tested from 2m for impact resistance

LSTM Classifier
class ImprovedLSTMAutoencoder(nn.Module):
    def __init__(self, input_dim=4, hidden_dim=64, num_layers=2, dropout=0.1):
        super().__init__()
        self.encoder = nn.LSTM(
            input_dim, hidden_dim, num_layers,
            batch_first=True, dropout=dropout, bidirectional=True
        )
        self.bottleneck = nn.Sequential(
            nn.Linear(hidden_dim * 2, hidden_dim),
            nn.ReLU(),
            nn.Dropout(dropout),
            nn.Linear(hidden_dim, hidden_dim // 2),
            nn.ReLU(),
            nn.Linear(hidden_dim // 2, hidden_dim)
        )
        self.decoder = nn.LSTM(
            hidden_dim, hidden_dim, num_layers,
            batch_first=True, dropout=dropout
        )
        self.output_layer = nn.Linear(hidden_dim, input_dim)

    def forward(self, x):
        batch_size, seq_len, _ = x.shape
        encoder_out, _ = self.encoder(x)
        final_representation = encoder_out[:, -1, :]
        compressed = self.bottleneck(final_representation)
        decoder_input = compressed.unsqueeze(1).repeat(1, seq_len, 1)
        decoder_out, _ = self.decoder(decoder_input)
        reconstructed = self.output_layer(decoder_out)
        return reconstructed, compressed
